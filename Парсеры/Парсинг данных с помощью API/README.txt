1. API HeadHunter

Напишите код, который соберет следующие данные о 1000 работодателей Алтайского края, которые имеют на текущий момент открытые вакансии: id, название работодателя, ссылки на вакансии, количество открытых вакансий.
Для выполнения задания не нужно регистрировать приложение и использовать токен.
→ Данные должны быть отсортированы по количеству вакансий (от большего к меньшему) и собраны в csv файл.
